# ECMAScript 6 入门

《ECMAScript 6 入门》是一本开源的 JavaScript 语言教程，全面介绍 ECMAScript 6 新引入的语法特性。

来自 <https://github.com/ruanyf/es6tutorial>， 添加了自己的学习笔记。

依赖 <https://github.com/wuliupo/ditto>
